# 내 약 노트 사용 설명서

복약을 빠뜨리지 않도록 도와주는 앱 **내 약 노트**의 사용법입니다.
약과 영양제를 등록하고, 매일 복용을 체크하고, 정해진 시간에 LINE으로 알림을 받을 수 있습니다.

---

## 1. 시작하기

### 앱 열기 / 설치
- 브라우저에서 앱 주소를 엽니다.
- **홈 화면에 추가**하면 일반 앱처럼 사용할 수 있습니다.
  - iPhone(Safari): 공유 버튼 → "홈 화면에 추가"
  - Android(Chrome): 메뉴 → "앱 설치" / "홈 화면에 추가"

### 언어 바꾸기
- 화면 **오른쪽 위의 국기**(🇯🇵 / 🇰🇷)를 누르면 일본어 / 한국어를 선택할 수 있습니다.
- 선택한 언어는 자동으로 저장됩니다.

### 화면 구성(아래 탭)
| 탭 | 내용 |
|----|------|
| **오늘** | 오늘 먹을 약을 체크합니다. |
| **약** | 약·영양제를 등록·수정합니다. |
| **알림** | 복약 시간(리마인더)을 설정합니다. |
| **병원용** | 약 목록을 정리해 인쇄/PDF로 저장합니다. |
| **설정** | Google 로그인, LINE 연동, 데이터 관리. |

---

## 2. 약 등록하기

1. 아래의 **약** 탭을 누릅니다.
2. 오른쪽 위 **＋ 추가** 를 누릅니다.
3. 정보를 입력합니다.
   - **종류**: 💊 약 / 🌿 영양제
   - **이름**: 예) 암로디핀정 5mg
   - **복용량**: 예) 1정
   - **식사와의 관계**: 지정 안 함 / 식전 / 식후 / 식간
   - **복용 시점**: 아침 / 점심 / 저녁 / 자기 전 (여러 개 선택 가능)
   - **메모(선택)**: 주의사항 등
4. **저장하기** 를 누릅니다.

> 수정·삭제하려면 약 목록에서 해당 약의 ✏️ 아이콘을 누르세요.

### 사진으로 약 자동 등록하기 (AI 인식)
1. **약** 탭에서 **📷 사진에서 AI로 읽기** 를 누릅니다.
2. **① 읽을 사진의 종류** 를 선택합니다.
   - **💊 약 사진**: 약 봉투·포장·알약 사진
   - **📋 약 목록 텍스트**: 처방전·약 목록표·약 수첩 등 글자가 적힌 사진
3. **📷 사진 선택** 을 누르면 **촬영 / 사진 보관함 / 파일** 중에서 고를 수 있습니다.
4. **🔍 AI로 읽기** 를 누르면 AI가 약 정보를 읽어 줍니다.
5. 인식된 약 중 등록할 항목을 체크하고 **선택한 약 등록하기** 를 누릅니다.

> AI 인식 결과는 **선택한 언어**(한국어/일본어)로 표시됩니다. 인식이 정확하지 않을 수 있으니 등록 후 내용을 확인해 주세요.

---

## 3. 오늘 복용 체크하기

1. **오늘** 탭을 엽니다.
2. 시간대별(아침/점심/저녁/자기 전)로 약이 카드로 표시됩니다.
   - 약은 💊, 영양제는 🌿 아이콘으로 구분됩니다.
   - 같은 시간대의 약과 영양제는 **하나의 카드**에 구분선으로 나뉘어 표시됩니다.
3. 먹은 약은 **복용함** 버튼을 눌러 ✓ 표시합니다.
4. 그 시간대를 한 번에 처리하려면 **한꺼번에 복용** 을 누릅니다.
5. 위쪽 원형 그래프로 오늘의 **달성률**을 확인할 수 있습니다.

> 날짜 위의 `‹` `›` 로 다른 날짜의 기록도 확인·수정할 수 있습니다.

---

## 4. 알림(리마인더) 설정

1. **알림** 탭을 엽니다.
2. **＋ 추가** 를 누릅니다.
3. **시간**(예: 09:00)과 **시점**(아침/점심/저녁/자기 전)을 정합니다.
4. **저장하기** 를 누릅니다.

설정한 시간이 되면 알림을 받습니다.
- **LINE 알림**: LINE을 연동하면 **앱을 닫아도** 휴대폰으로 알림이 옵니다(아래 5번 참고). 30분 간격으로 확인하므로 실제 도착은 다소 지연될 수 있습니다.
- **브라우저 알림**: 알림을 허용하면 기기에서도 알림이 표시됩니다.

---

## 5. LINE으로 알림 받기

앱을 닫아 두어도 복약 시간에 LINE으로 알림을 받을 수 있습니다.

1. **설정** 탭을 엽니다.
2. **LINE으로 알림 받기** 항목에서 **LINE과 연동하기** 를 누릅니다.
3. LINE 로그인 화면에서 로그인하고, 안내에 따라 **친구 추가** 를 합니다.
4. 연동이 완료되면 설정 화면에 **LINE 연동됨** 으로 표시됩니다.

> - iPhone에서는 로그인 화면이 Safari에서 열릴 수 있습니다. 로그인 후 앱으로 돌아오면 자동으로 연동이 반영됩니다.
> - 연동을 끊으려면 **LINE 연동 해제** 를 누르세요.

---

## 6. Google 로그인 & 기기 간 동기화

Google 계정으로 로그인하면 데이터가 본인의 Google 드라이브에 저장되어, **다른 기기에서도 같은 데이터**를 사용할 수 있습니다.

1. **설정** 탭 → **Google로 로그인** 을 누릅니다.
2. Google 계정으로 로그인합니다.
3. 이후 데이터가 자동으로 동기화됩니다(수동으로 하려면 **☁️ 지금 동기화**).
4. 로그아웃하려면 **로그아웃** 을 누릅니다.

> 로그인하지 않아도 사용할 수 있지만, 데이터는 **이 기기에만** 저장됩니다.

---

## 7. 병원용 요약 (인쇄 / PDF)

진료 시 보여줄 수 있도록 약 목록을 정리합니다.

1. **병원용** 탭을 엽니다.
2. 등록된 **약 목록**(영양제 제외)이 표시됩니다.
3. **🖨 인쇄 / PDF 저장** 을 누르면 인쇄하거나 PDF로 저장할 수 있습니다.

---

## 8. 데이터 관리

- **설정** 탭의 **데이터** 항목에서 관리합니다.
- **모든 데이터 삭제**: 등록한 약·기록을 모두 삭제합니다. (되돌릴 수 없으니 주의)

---

## 9. 자주 묻는 질문

**Q. 알림이 오지 않아요.**
- 알림 권한이 허용되어 있는지 확인하세요.
- LINE 알림은 LINE 연동과 **친구 추가**가 모두 되어 있어야 합니다.
- LINE 알림은 30분 간격으로 확인하므로 설정 시각보다 조금 늦게 올 수 있습니다.

**Q. 다른 휴대폰에서도 쓰고 싶어요.**
- 두 기기 모두 같은 **Google 계정으로 로그인**하면 데이터가 동기화됩니다.

**Q. 약과 영양제는 어떻게 구분되나요?**
- 등록 시 **종류**에서 선택하며, 오늘 화면에서 💊(약) / 🌿(영양제) 아이콘으로 표시됩니다.

**Q. 사진 인식이 정확하지 않아요.**
- 글자가 선명하게 나오도록 다시 촬영하거나, "약 목록 텍스트" 모드를 사용해 보세요. 등록 후 내용을 직접 확인·수정하는 것을 권장합니다.

---

문제가 계속되면 앱 관리자에게 문의해 주세요.
